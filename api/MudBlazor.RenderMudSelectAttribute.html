    Syntax
    [AttributeUsage(AttributeTargets.Property)]
public class RenderMudSelectAttribute : FormGeneratorAttribute
    Remarks

    Decorating a System.String property with this attribute causes the form generator to render the property as a MudBlazor.MudSelect`1 component.

    This attribute is only valid when placed on a property that returns a System.String value.

    The options, for the select should be specified using the Options parameter, with a comma separated list of string values.
