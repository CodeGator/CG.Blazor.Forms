    Syntax
    [AttributeUsage(AttributeTargets.Property)]
public class RenderMudColorPickerAttribute : FormGeneratorAttribute
    Remarks

    Decorating a MudBlazor.Utilities.MudColor property with this attribute causes the form generator to render the property as a MudBlazor.MudColorPicker component.

    This attribute is only valid when placed on a property that returns a MudBlazor.Utilities.MudColor value.
