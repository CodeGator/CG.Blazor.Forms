    Syntax
    [AttributeUsage(AttributeTargets.Property)]
public class RenderMudSliderAttribute : FormGeneratorAttribute
    Remarks

    Decorating a numeric property with this attribute causes the form generator to render the property as a MudBlazor.MudSlider`1 component.

    This attribute is only valid when placed on a property that returns a numeric value (in other words, an int, long, byte, float, double, or decimal type).
