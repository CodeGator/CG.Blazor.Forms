public class DataAnnotationsValidatorAttribute : FormGeneratorAttribute
    Remarks

    Decorating the top-level model's class with this attribute causes the form generator to render a Microsoft.AspNetCore.Components.Forms.DataAnnotationsValidator inside the generated form.

    This attribute is only valid when placed on the class definition for the top-level model class.
