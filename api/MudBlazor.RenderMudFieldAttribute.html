    Syntax
    [AttributeUsage(AttributeTargets.Property)]
public class RenderMudFieldAttribute : FormGeneratorAttribute
    Remarks

    Decorating a System.String property with this attribute causes the form generator to render the property as a MudBlazor.MudField component.

    This attribute is only valid when placed on a property that returns a System.String value.
