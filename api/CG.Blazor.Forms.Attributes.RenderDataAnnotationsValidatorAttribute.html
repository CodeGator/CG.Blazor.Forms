    Class RenderDataAnnotationsValidatorAttribute

    This class is an attribute that, when applied to a class, causes the form generator to render a Microsoft.AspNetCore.Components.Forms.DataAnnotationsValidator tag, within the form.

    Syntax
    [AttributeUsage(AttributeTargets.Class)]
public class RenderDataAnnotationsValidatorAttribute : FormValidationAttribute
    Remarks

    This attribute is only valid when placed on the class definition for the top-level model class.

    Examples

    Here is an example of decorating a model class to render a data annotations validator:

    using CG.Blazor.Forms.Attributes;

[RenderDataAnnotationsValidator]
class MyModel
{

}
