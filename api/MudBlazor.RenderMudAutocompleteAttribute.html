    [AttributeUsage(AttributeTargets.Property)]
public class RenderMudAutocompleteAttribute : FormGeneratorAttribute
    Remarks

    Decorating a string property with this attribute causes the form generator to render the property as a MudBlazor.MudAutocomplete`1 component.

    This attribute is only valid when placed on a property that returns a string value.

    In order for the search operation to function, in the auto-complete control, the name of a publicly accessable search function must be placed into the SearchFunc property. That search function can either live on the same object as the decorated property, or, it can live on the top-level model for the form, which can then be thought of as a kind of view-model.
